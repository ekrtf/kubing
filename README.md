kubing
======

Just messing about with k8 and other fun stuff.

## Quickstart

```sh
docker run -p 1111:1111 -d ekrtf/ocean:kubing_v0.0.1
```

## Slowstart

```sh
git clone https://github.com/ekrtf/kubing.git
cd kubing/app
npm install
npm start
```
